# TSAB Website Brief

## Chosen product direction

TSAB should be built as a **modern student community portal**.

That means the site should feel:
- useful first
- community-led
- youth-oriented
- clean and premium
- easy to update

It should **not** feel like:
- a static brochure
- a poster dump
- a committee-only internal page
- a chat archive

## Core website goal

Create a modern, trustworthy website for TSAB that helps Telugu students in Bhopal:

- find their community quickly
- discover events and celebrations
- access opportunities and useful updates
- celebrate member achievements
- browse media memories
- understand how to join and contribute

## Product lens

TSAB is not just an event club.
From the uploaded group exports, TSAB already behaves like a real student network with:

- community onboarding
- opportunity sharing
- volunteer coordination
- media collection
- achievement amplification

The public website should mirror that structure.

## Public navigation

- Home
- About TSAB
- Events
- Opportunities
- Achievers
- Gallery
- Working Groups
- Team
- Join / Contact

## Primary homepage message

**A home for Telugu students in Bhopal — community, culture, opportunities, and growth.**

## What should make TSAB stand out

Many student association sites stop at:
- event posters
- gallery images
- a committee list

TSAB should stand out by emphasizing:
- opportunities
- achiever stories
- useful student support
- visible community pathways
- clear working groups

## Visual identity direction

Observed from the shared posters and logo assets:

- deep royal blue should remain the dominant brand color
- gold should remain the accent color
- the logo should stay prominent
- Telugu cultural motifs should be used very lightly
- the UI should feel modern, not ceremonial-heavy

## Recommended design system

### Colors
- brand primary: deep royal blue
- accent: gold / saffron gold
- neutral background: white / soft cream
- text: deep slate / near-black
- soft surfaces: pale blue-gray

### UI style
- rounded but not overly bubbly cards
- generous spacing
- section-based homepage
- clean icons
- subtle gradients only where useful
- restrained decorative dividers

### Motion
- subtle only
- small fade-ins, hover lifts, and panel transitions
- no flashy festival-style animation

## Homepage blocks

1. Hero
2. What TSAB offers
3. Upcoming event
4. Opportunities snapshot
5. Featured achiever
6. Gallery preview
7. Working groups / community pathways
8. Join TSAB CTA

## Working groups to show publicly

Instead of embedding WhatsApp chats, frame the four groups as public functions:

- Main TSAB Community
- TSAB Forum & Opportunities
- Volunteer / Social Media Cell
- Media Cell

## Editor experience goal

A future team member with no coding experience should be able to:

- add an event
- upload a gallery image
- update a team member
- add an achiever story
- publish an opportunity
- update contact / join links

using documentation and browser-based editing first, with optional CMS later.
